# AutoI v2
A simple trading bot project.

## Goal

- Build a clean trading bot project
- Use it for portfolio
- Test with very small money
- Focus on system and learning

## Project Parts

### data
Get market price data.

### strategy
Define buy and sell rules.

### backtest
Test the strategy on old data.

### execution
Send real buy and sell orders.

### logging
Save trade logs and error logs.

## First Step

- Get BTC/USDT price data
- Save data to CSV
- Make one simple backtest
